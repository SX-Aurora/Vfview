VFD file format

What does VFD even mean?
HEADER:
Size [Bytes] Name Description
4 vfd_version VFD version the file has been created under
16 vftr_fileid Individual ID of vfd file
24 date Date at which the vfd file has been created
8 vftr_interval Sampling time interval in microseconds
4 threads Number of OpenMP threads
4 thread Associated OpenMP therad
4 tasks Number of MPI ranks
4 task Associated MPI rank
8 cycletime CPU cycle time in (unit?) as measured by vftrace
8 inittime Time of first sample in (unit?)
8 runtime Total runtime in (unit?)
4 samplecount Number of registered samples
4 stackscount Number of registered stacks
4 stacksoffset File position at which stack data begins
4 sampleoffset File position at which sample data begins
4 reserved Unclear
4 mpi_info_dim Length of the array containing the MPI data types
STACK INFO:
Size [Bytes] Name Description
4 id Stack ID
4 levels Number of functions in the stack
4 caller Stack ID of the calling function
4 len Length of the succeeding function name string (in bytes)
len rec Name of the function at the top of the stack
SAMPLE INFO:
Size [Bytes] Name Description
4 sidw Entry type of this sample (entry, exit or message)
4 stackID ID of the associated stack
8 ldelta Time spent in this sample in (units?)
MESSAGE INFO:
Size [Bytes] Name Description
4 sidw Entry type of this sample (entry, exit or message)
4 dir Message direction (send / receive)
4 mpi_info_index Index of the datatype in the internal MPI type matrix
4 peer Rank of the sending process(?)
8 tstart Start time of message in (units?)
8 tstop End time of message in (units?)
4 count Number of messages send
4 tag Message tag
4 type MPI datatype
4 data_type_size Size of (custom) data type
